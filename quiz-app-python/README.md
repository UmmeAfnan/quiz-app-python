# 🧠 Python Quiz App

## 📌 Features

- ✅ 5 quiz questions with instant feedback
- ⏱️ Timer to track how long the quiz takes
- 🔁 Retry logic (optional)
- 📊 Scoreboard logic (expandable)
- 🧠 Beginner-friendly logic using `functions`, `if`, `input()`, and more
